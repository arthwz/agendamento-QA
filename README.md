# Agendamento-QA

Nome do projeto: API-Agendamento-QA.

Integrantes: Filipe Cabral Monteiro, João Lucas Santos Oliveira, Bruno Maximino Reis, Felipe dos Santos de Oliveira, Arthur Carvalho Santos.

Tema escolhido: Sistema de Agendamento (API simples em Python).

Descrição inicial: API desenvolvida em Python (FastAPI/Flask) para gerenciamento de agenda de limpeza. O projeto foca na criação de horários, verificação de conflitos e cancelamentos, com o objetivo principal de demonstrar a aplicação de testes de API (manuais e automatizados usando pytest e Postman) e o registro das evidências de qualidade.
